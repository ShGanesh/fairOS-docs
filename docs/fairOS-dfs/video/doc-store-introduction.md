#### Contact Database

- User Login (user1)
- Pod Open (pod1)
- Create Table (doc create contact first_name=string,age=number)
- Open Table (doc open contact)
- Add Entry (doc put contact ...)
- Count Entries
- Get Entry (by id)
- Find Entry (by first_name)
- Find Entry (greater than a certain age)
